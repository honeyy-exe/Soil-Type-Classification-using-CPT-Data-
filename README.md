# Soil Type Classification using CPT Data

Description
Usually I’m coding apps, but now I’m talking to dirt. This project uses Machine Learning with a Random Forest model to predict soil types from Cone Penetration Test (CPT) data. Because even civil engineers need AI to understand soil.

Dataset

CPT data with cone resistance, sleeve friction, and other measurements

Labels: Soil types (Clay, Silt, Sand, etc.)

Tech Stack

Python

Pandas & NumPy

Scikit-learn (Random Forest)

Matplotlib & Seaborn

Features & Outcome

Predicts soil type based on CPT measurements

Shows which soil traits matter most

Turns boring CPT numbers into actionable insights

Fun Fact
Even soil has secrets—thankfully, Random Forest is here to spill them. 🌱
